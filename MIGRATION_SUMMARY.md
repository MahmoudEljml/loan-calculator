## 🎯 النتيجة النهائية:

### المسار 1️⃣ - من صفحة التعديل:
```
عندما يضغط المستخدم على زر "الملاحظات" في صفحة التعديل:
  1. يتم فتح Sheet من الجانب الأيمن
  2. يرى جميع الملاحظات الحالية
  3. يمكنه إضافة ملاحظة جديدة
  4. يمكنه حذف ملاحظة موجودة
  5. يبقى في نفس الصفحة (لا توجد عملية إعادة تحميل)
```
